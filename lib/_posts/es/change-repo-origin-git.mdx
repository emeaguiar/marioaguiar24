Más de alguna vez he necesitado **cambiar el origen de un repositorio**, así que para 
hacerme la vida más fácil decidí hacer la anotación aquí.

Lo primero que se necesita es un nuevo repositorio, digamos que es `git@github.com:emeaguiar/mah-theme.git`

Ahora, la cosa es tan simple como eliminar el viejo remoto de origen, agregar el nuevo, y configurarlo como master.

<Code
    language="bash"
    children={`# Eliminar viejo origen
~ git remote rm origin

# Agregar nuevo origen
~ git remote add origin git@github.com:emeaguiar/mah-download-manager.git

# Configurarle como master
~ git config master.remote origin`}
/>

¡Listo! Ahora al ejecutar `git remote get-url origin --all` verán el nuevo repositorio.
